# NexaAI-Pro-Enterprise-AI-Dashboard

A modern, responsive AI dashboard UI built with Tailwind CSS, Lucide Icons, and Chart.js.  
Designed for enterprise-level AI monitoring and metrics visualization.

## Features
- Responsive design with Tailwind CSS
- Interactive charts using Chart.js
- Sidebar navigation and search functionality
- Cards and widgets for metrics
- Smooth animations and hover effects

## Tech Stack
- HTML, CSS, JavaScript
- Tailwind CSS
- Chart.js
- Lucide Icons
- Google Fonts (Inter)

## Usage
Simply open `index.html` in your browser. Ready to integrate with backend APIs.
